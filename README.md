# 🚀 AgeIS-X

### AI-Powered Phishing & Fraud Detection Platform

AgeIS-X is an intelligent cybersecurity SaaS platform built to detect phishing URLs and online fraud attempts in real time using Machine Learning and NLP techniques.

It combines modern frontend engineering with scalable backend architecture to deliver fast, accurate, and production-ready threat analysis.

---

## 🔐 Why AgeIS-X?

Phishing attacks are growing rapidly and traditional rule-based systems fail to detect sophisticated threats.

AgeIS-X solves this by using:

* 🧠 Machine Learning–based URL classification
* 🔎 NLP techniques (TF-IDF, pattern analysis)
* ⚡ Real-time risk scoring
* 📊 Interactive threat visualization dashboard

---

## 🛠️ Local Development

Clone and run locally:

```bash

cd ageis-x
npm install
npm run dev
```

App will run at:

```
http://localhost:3000
```
